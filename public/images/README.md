# Images Directory

This directory contains images for the website:

- `PRAB_24_041002_FMA_CC.png` - Feature image from Crab Crossing paper
- `people/` - Team member photos
- `projects/` - Project images
- `og-image.png` - Open Graph image for social sharing

## To add your images:

1. Copy `PRAB_24_041002_FMA_CC.png` from your existing site's assets folder
2. Add team member photos to `people/` directory (recommended size: 400x400px)
3. Create an `og-image.png` for social media previews (recommended: 1200x630px)
